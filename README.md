# Samizdat-Plugin-Mailer

Email list/campaign mailer for [Samizdat](https://fakenews.com). Extracted from the monorepo with
history; requires core **Samizdat** (PERL5LIB or installed).

    perl Makefile.PL && make && make test    # core on PERL5LIB
    make install

Enable via `extraplugins: [Mailer]`.
